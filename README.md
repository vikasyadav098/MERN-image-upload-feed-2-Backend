# 🚀 MERN Image Upload Feed — Backend API

A powerful Node.js & Express backend that handles image uploads, post creation, and feed retrieval for the MERN Image Upload Feed application.

---

## ✨ Features

- 📤 Image Upload using ImageKit
- 📝 Create Post API
- 📰 Fetch Feed API
- 🌐 CORS enabled for frontend
- 🗄️ MongoDB Database
- 🔐 Environment-based configuration
- ☁️ Deployed on Render

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- ImageKit
- CORS
- Dotenv
- Render (Backend Hosting)
